HexoBlog

安装准备

  • git&github账户
  • node.js
  • 文本编辑器

node.js安装(windows10环境)

  • node. js官方网站https://nodejs.org 下载环境
  • 本人使用的是zip文件非安装版本，下载解压之后进行系统环境变量的配置。
  • 新建环境变量
  • 变量名:NODE_PATH
  • 变量值:解压路径(E:\nodejs)
  • 再在path中添加%NODE_PATH%属性值即可
  • 在cmd控制台中可使用node -v 命令进行版本查看
  • 同时包含有npm包管理器，同样使用npm -v 命令查看版本

npm国内镜像插件工具cnpm安装

  • 由于网络问题，npm安装速度缓慢，使用taobao镜像进行安装
  • 安装命令:npm install -g cnpm –registry=https://registry.npm.taobao.org
  • 安装完成后同样使用cnpm -v 查看该工具版本

hexo安装

  • 前面的步骤完成之后就可以进行安装，安装命令
  • cnpm install -g hexo-cli
  • 安装完成后同样使用hexo -v 命令验证版本

hexo生成

  • 首先在任意路径下创建一个文件夹，该文件夹用于存放所搭建hexoBlog的文件
  • 命令行终端进入该文件夹即可执行以下命令来初始化blog：
  • hexo init
  • 以上这一步骤需要有git环境作为准备，否则会出现错误

hexoBlog启动

  • 执行命令hexo s即可
  • 浏览器中访问地址localhost:4000进入页面

hexoBlog使用

  • 在hexo根目录下使用命令hexo n “BlogName”即可，hexo会创建Blog文章到路径source/_posts/下
  • 或者命令行终端进入hexo根目录下的文件夹~/source/_posts/,直接在该目录下创建Blog文章
  • 创建文章完成之后回到根目录下按步骤执行以下命令
  • hexo clean 清理
  • hexo g 生成
  • 再启动hexo即可

留个坑

  • 将搭建好的blog部署在github上
